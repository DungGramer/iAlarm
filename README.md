# iAlarm
Simple alarm for android 4.4 or higher

## Features
✅ Set alarm

## Usage
Download [iAlarm.apk](https://raw.githubusercontent.com/DungGramer/iAlarm/master/iAlarm.apk). Enable the “Unknown Sources” setting and install APK.
